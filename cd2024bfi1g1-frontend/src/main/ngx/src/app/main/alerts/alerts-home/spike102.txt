INDICAR CON UN NUMERO EN LA PESTAÑA DEL NAVEGADOR

  Actualmente, el sistema muestra la cantidad de filas en la tabla de alertas en el título de la página.

  El problema es q para actualizar el título con la cantidad de alertas, el usuario debe entrar en la vista de alertas. Este comportamiento no es el deseado, ya que queremos que el título se actualice constantemente, independientemente de si el usuario está en la vista de alertas o no.

  SOLUCION PROPUESTA
  Una posible solucion consiste en crear un servicio en el componente principal (Main) que se encargue de realizar la consulta periódica que devuelve la cantidad de alertas disponibles. Este servicio debe estar ejecutándose de forma continua y actualizando el título cada cierto tiempo, sin necesidad de que el usuario esté en la vista de alertas.

  CONSIDERACIONES
  Gestión del temporizador: Es importate destruir este temporizador cuando el usuario haga logout de la página para evitar que la consulta siga realizándose de manera indefinida en segundo plano.
